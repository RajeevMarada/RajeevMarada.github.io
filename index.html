ARCHIVES

RISC-V ECG Accelerator

Heterogeneous SoC Design

Engineered a heterogeneous SoC integrating a RISC-V processor and a custom neural network via AXI Stream, achieving 92% accuracy in real-time ECG classification. Validated the complete design through FPGA deployment.

  • RISC-V
  • AXI Stream
  • Neural Networks
  • FPGA

This project was the culmination of my IISc certification, synthesizing system architecture, hardware design, and machine learning. The goal was to accelerate ECG classification for real-time, portable medical devices.

We designed a custom neural network accelerator in Verilog and integrated it with a RISC-V core using the AXI Stream protocol for high-throughput data transfer. The system achieved 92% accuracy through Q1.15 fixed-point arithmetic optimization, balancing performance with resource constraints.

Neural Net RTL Engine

Hardware Inference

Designed a Verilog RTL 3-layer neural network inference engine with ROM-based weights, achieving over 85% accuracy on the Semeion dataset. Performed full simulation, synthesis, and implementation in Xilinx Vivado.

  • Verilog RTL
  • Neural Networks
  • Vivado

This was a deep dive into implementing AI directly in hardware. I designed a 3-layer neural network from scratch in Verilog, complete with custom ROM-based weights and a FIFO input buffer. The entire engine was verified, synthesized, and implemented in Xilinx Vivado, successfully passing timing closure and proving its readiness for FPGA deployment.

APB UVM Testbench

Advanced Verification

Architected a modular UVM testbench with reusable master/slave agents to rigorously validate APB protocol compliance. Achieved 100% functional and code coverage and confirmed design robustness with complete assertion coverage.

  • UVM
  • SystemVerilog
  • 100% Coverage

This project was a practical, industry-focused dive into advanced verification. I learned to think like a verification engineer, building a complete testbench from scratch with drivers, monitors, scoreboards, and assertions. Achieving 100% functional and code coverage required systematic test planning and creative stimulus generation to hit every edge case.

SEC-DED-DAEC Module

Data Integrity (IEEE Published)

Led development of a novel error-correcting module, optimizing reversible logic to achieve a 5.67% power reduction and 4.52% delay improvement over other designs, culminating in an IEEE publication.

  • Error Correction
  • Low-Power
  • IEEE Published

This was my final year project, born from the question: "How do we handle burst errors?" We designed a comprehensive Single Error Correction, Double Error Detection, and Double Adjacent Error Correction (SEC-DED-DAEC) architecture.

A key milestone was integrating this completed module with the **AMBA AHB-APB bridge** I had built during my Maven Silicon internship. This connected a complex academic project to a practical, industry-standard component, proving its system-level applicability.

Hybrid Hack 2021 Finalist

Cross-Domain Innovation

Co-developed "Charge On Go," a novel vehicle-mounted hybrid power system. My role involved simulating the system's electrical performance, projecting an annual energy generation of 465.15 kWh per unit.

  • Python
